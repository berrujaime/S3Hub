Privacy Policy

Last updated: May 30, 2026

This privacy policy applies to the S3Hub mobile application ("the App"), developed by Jaime Berruete ("BerruApps").

Data Collection and Storage

S3Hub is a cloud storage client that connects to Amazon S3 and Storj services. The App does not collect, transmit, or store any personal data on external servers controlled by the developer.

All data is stored exclusively on your device:

  • S3/Storj Credentials (Access Key and Secret Key): Stored locally in the device's encrypted storage (SecureStore) and used solely to authenticate with your own S3-compatible storage services.
  • Connection Preferences (selected bucket, language, preview settings): Stored locally on your device.
  • File Cache: Image and video thumbnails are cached locally in the app's cache directory to improve browsing performance. This cache is cleared automatically after 7 days or when the app goes to the background.
  • File Listings Cache: Directory listings are cached locally to reduce API calls. Expires after 7 days.

Data Sharing

S3Hub does not share any data with third parties. All network communication occurs directly between your device and your configured S3/Storj endpoints. No data passes through intermediary servers controlled by the developer.

Permissions

The App may request the following device permissions:

  • Notifications: To notify you when file uploads complete.
  • Media Library / Storage: To save downloaded files to your device's gallery.
  • File Access: To allow you to select files for upload to your S3 buckets.

Data Security

Sensitive credentials are stored using the platform's encrypted storage (iOS Keychain / Android EncryptedSharedPreferences via expo-secure-store). API communications use HTTPS/TLS encryption.

Data Retention and Deletion

  • All stored data can be deleted by uninstalling the App.
  • Individual connections can be deleted from within the App's interface.
  • Cache data is automatically cleaned when the app goes to the background.

Third-Party Services

The App connects to the following services at your direction:

  • Amazon Web Services S3 (Privacy Policy)
  • Storj DCS (Privacy Policy)

Children's Privacy

The App does not knowingly collect personal information from children under 13.
